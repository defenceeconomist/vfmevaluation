# Release Notes

This page records release-level changes to the Value for Money Evaluation Knowledge Base.

## Site Structure Update - 2026-06-29

### Removed

- Removed the Reports section from the published knowledge base navigation and static site output.

## Initial Release - 2026-05-30

Initial public release of the knowledge base.

### Added

- Source-note library for value for money evaluation, including OPM, Verian Value for Investment, Green Book, Magenta Book, JSP 507, NAO, OECD DAC, DfT, Government Efficiency Framework, and sourcing guidance.
- Interactive notes map showing relationships between notes, citations, and linked sources.
- Shared BibTeX bibliography for source traceability.
- Quarto website structure with notes, search, sidebars, and rendered HTML outputs.

### Included Source Areas

- Central government appraisal, evaluation, and efficiency guidance.
- King, OPM, and Verian Value for Investment guidance.
- OECD DAC evaluation criteria and VfI relationship notes.
- Department for Transport value for money and appraisal material.
- Sourcing and consultancy playbook material for VfM rubric creation.

### Notes

- This is the baseline release. Future entries should describe user-facing additions, substantial content revisions, structural site changes, source updates, and rendered-output changes that matter to readers.
